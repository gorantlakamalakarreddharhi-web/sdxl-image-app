# SDXL Image Generator & Editor

## Quick Start
1. Get HF token: https://huggingface.co/settings/tokens
2. `cp .env.example .env` and add `HF_API_KEY=`
3. `npm run dev`

## Setup

npm install
cp .env.example .env # add HF_API_KEY and NEXT_PUBLIC_API_BASE
npm run dev

Open http://localhost:3000 and test /api/generate and /api/edit with your Hugging Face SDXL token.
